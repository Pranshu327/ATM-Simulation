# ATM Simulation System (C++)

## Description
This is a simple ATM simulation project written in C++.
It works on the terminal and uses basic programming concepts.

## Features
- 4-digit PIN verification
- Maximum 3 wrong PIN attempts
- Check balance
- Deposit money
- Withdraw money
- Logout option
- Exit option
- Menu-driven program

## Initial Details
- Initial Balance: Rs. 1000
- Default PIN: 1234

## Concepts Used
- Functions
- Global variables
- Loops
- Conditional statements
- Switch case
- Input / Output

## Menu Options
1. Check Balance  
2. Deposit Money  
3. Withdraw Money  
4. Logout  
5. Exit  

## How It Works
1. User enters ATM PIN.
2. After verification, menu is shown.
3. User selects an option.
4. Program runs until Exit is chosen.
